# Excepciones de Estilo y Contenido (Controladas)

**Regla:** cualquier desviación del estándar definido en `docs/ESTILO_00_guia_tono_y_voz_equipo.md` u otras normativas (contenido, términos prohibidos) debe registrarse aquí ANTES de aplicarse. El validador de estilo ignorará las alertas si el documento incluye el ID de excepción correspondiente.

---

## 📝 Plantilla de Excepción
Para registrar una nueva excepción, copia, pega y rellena el siguiente bloque al final de este documento:

```markdown
### [Título de la Excepción]
- **ID:** EXC-YYYY-MM-DD-XX
- **Fecha:** YYYY-MM-DD
- **Archivo afectado:** [Ruta del archivo]
- **Regla excepcionada:** (ej. voz institucional / tono / promesas absolutas / terminología DUA)
- **Motivo:** [Por qué se permite esta desviación en este caso concreto]
- **Alcance:** [Qué se permite exactamente y qué sigue estrictamente prohibido]
- **Aprobado por:** Equipo ZAC
```
