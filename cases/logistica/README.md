# Caso de Prueba: Logística FTL

Este directorio contiene la implementación específica del caso de prueba de logística, utilizado para validar la arquitectura agnóstica del sistema.

## 📁 Estructura del Caso

- **[output/bloque_1/](output/bloque_1/)**: Entregables estratégicos generados (Entregables 06, 07, 08).
- **[evidencias/](evidencias/)**: Reportes de ejecución y validación (Dry Run 2).

## ⚠️ Nota de Mantenimiento
Este caso se encuentra actualmente en fase de **Aislamiento por Copia**. Las rutas maestras en la raíz del repositorio (`output/bloque_1/`) siguen existiendo para no romper los scripts de validación actuales, pero la fuente de verdad del caso se está trasladando progresivamente a este directorio.
